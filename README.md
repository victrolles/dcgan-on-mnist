# dcgan-on-mnist

### Setup

Create venv
```bash
python -m venv .venv
```

Activate venv
```bash
source .venv/bin/activate
```

Install requirements
```bash
pip install -r requirements.txt
```

Install pytorch with cuda
```bash
pip install torch torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/cu118
```
